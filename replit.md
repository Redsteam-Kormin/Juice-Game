# Brewery Simulator (Juice Game)

## Overview
A Python-based terminal game where players manage a brewery that makes and sells juice. The game features:
- Juice brewing and selling mechanics
- Mental health system
- Random events (investors, new juice discoveries)
- Ranking system based on earnings
- Simple text-based interface

## Project Information
- **Language**: Python 3.11
- **Type**: Terminal/Console game
- **Main File**: game.py
- **Dependencies**: None (uses only Python standard library)

## How to Run
The game runs as a console application. Players interact via text input to:
- Name their brewery
- Sell juice to make money
- Drink juice to maintain mental health
- Skip days to progress
- Build up their brewery rank

## Architecture Notes
- Single-file Python script with global state
- No external dependencies required
- Uses terminal clearing and text-based UI
- Random events add variability to gameplay
